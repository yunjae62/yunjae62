- `React native`, `Lambda`, `DynamoDB`를 이용한 1인 프로젝트로 기획, 디자인, 개발, 운영, 마케팅의 전 과정을 경험함.

- 23.11 기준으로 약 2천회의 다운로드 수를 기록함.
